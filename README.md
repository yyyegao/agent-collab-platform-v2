# Agent协作平台 - 设计Demo

## 一、产品需求文档 (PRD)

### 1.1 产品定位
Agent协作平台是一款面向AI开发者和企业的可视化Agent管理及协作工具，通过类飞书的即时通讯界面，让用户能够直观地添加、配置和管理多个AI Agent，并支持群聊协作模式，实现多Agent协同工作。

### 1.2 核心价值
- **可视化操作** - 无需代码，通过拖拽/表单界面即可完成Agent的创建和配置
- **即时通讯体验** - 类飞书聊天界面，降低学习成本
- **零后端依赖** - 前端直接配置模型参数，无需部署后端服务
- **多Agent协作** - 支持群聊模式，多个Agent可以在同一空间内协同完成任务
- **灵活扩展** - 支持自定义Agent角色、Prompt和模型接入

### 1.3 核心功能 (P0必须实现)
| 功能 | 描述 |
|---|---|
| 聊天界面 | 类飞书的单聊/群聊界面，支持消息展示、发送、滚动加载 |
| Agent添加 | 添加新Agent，设置名称、头像、角色描述 |
| Agent配置 | 配置Agent的模型(API Key/Base URL)、System Prompt、温度等参数 |
| 单聊模式 | 1v1与单个Agent对话，获取响应 |
| 消息渲染 | 支持文本、代码块、Markdown渲染 |

### 1.4 页面结构
- 首页/对话列表 - `/`
- 聊天页 - `/chat/:id`
- Agent列表页 - `/agents`
- Agent配置页 - `/agents/new` 或 `/agents/:id/edit`
- 创建群聊页 - `/group/new`

### 5.3 拖拽聊天窗口（内容不变形）

使用 `@dnd-kit/core` 实现拖拽，核心是**拖拽容器与内容分离**：

```typescript
import { DndContext, DragOverlay } from '@dnd-kit/core';

// 拖拽外层框（可移动）
const DraggableWindow = ({ children }) => {
  const { attributes, listeners, setNodeRef, transform } = useDraggable({
    id: 'chat-window',
  });

  const style = {
    transform: transform 
      ? `translate(${transform.x}px, ${transform.y}px)` 
      : undefined,
    // 内容区域保持固定尺寸，不变形
    minWidth: '320px',
    maxWidth: '600px', 
  };

  return (
    <div ref={setNodeRef} style={style} {...listeners} {...attributes}>
      {/* 拖拽手柄 */}
      <div className="drag-handle">⋮⋮</div>
      
      {/* 内容区域 - 独立于拖拽 */}
      <div className="chat-content" style={{ transform: 'none' }}>
        {children}
      </div>
    </div>
  );
};
```

### CSS 保证内容不变形

```css
.chat-window {
  position: fixed;
  z-index: 1000;
  /* 宽度固定，内容不缩放 */
  width: 400px;
  min-width: 320px;
  max-width: 600px;
}

.chat-content {
  /* 内容独立布局 */
  display: flex;
  flex-direction: column;
  height: 100%;
  /* 拖拽时内容不变形 */
  transform: none !important;
  will-change: auto;
}

.message-list {
  flex: 1;
  overflow-y: auto;
  /* 禁用transform，防止变形 */
  transform: none !important;
}

.input-area {
  flex-shrink: 0;
  height: 60px;
}
```
